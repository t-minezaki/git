GakkenID 規約


第1条（定義）

GakkenID規約（以下「本規約」といいます）における用語の定義は以下のとおりとします。
  1. 「学研」とは、株式会社学研プロダクツサポート（以下「当社」といいます）を含む学研グループを構成する各会社をいいます。
  2. 「GakkenID」とは、「学研」が提供する本サービスおよび対象サービスを利用するために必要となるアカウントIDをいいます。ID保有者は、GakkenIDを利用することで、対象サービスに係るウェブサイトにログインすることができます。
  3. 「ID保有者」とは、本サービスに係るウェブページ上で当社の定める必要な情報を入力するID登録申込をし、当社によりGakkenIDの利用につき承認を受けた者をいいます
  4. 「ID登録申込」とは、GakkenIDの利用のための登録申込手続をいいます。
  5. 「本サービス」とは、「GakkenID」を発行・管理するサービスをいいます。
  6. 「対象サービス」とは、「GakkenID」を利用して受けられるインターネットを利用した情報提供、ショッピング、課金・決済サービスなどの学研が行う個々のサービスをいいます。対象サービスの詳細は対象サービスのページをご確認ください。

第2条（規約の適用）

  1. 本規約は、当社が定めるものであり、ID登録申込をしようとする者およびID保有者に対して適用されます。
  2. 当社は、当社の裁量により本規約の内容を変更することがあります。当社は、本規約の変更にあたり、変更後の本規約の効力発生日の1か月前までに、本規約を変更する旨、及び変更後の本規約の内容とその効力発生日を、当社ウェブサイト（URL:https://id.gakken.jp/）に掲示し、またはID保有者に電子メールで通知します。
  3. 当社が前項に基づき本規約の内容変更を行った後、ID保有者が本サービスまたは対象サービスを利用した場合、当該ID保有者は本規約の変更を承諾したものとみなします。
  4. 当社が本サービスに係るウェブページ上に掲載する諸注意等が存在する場合、当該諸注意等は本規約の一部を構成するものとします。

第3条（GakkenID保有者）

  1. ID登録申込後、当社がGakkenIDの利用につき承認した場合は、ID保有者として、対象サービスを利用できるものとします。
  2. ID保有者は、ID保有者となった時点で、本規約の内容に同意したものとみなされます。
  3. ID登録申込をしようとする者は、ID登録申込に際して、以下の条件を了承し、遵守するものとします。

    4. （1）本サービスに係るID保有者登録専用ページの入力欄に、自身のID登録に必要なデータを真実かつ正確に入力すること。代理による入力・ID登録申込は一切認められません。

    （2）ID登録申込時にメールアドレスを登録する際は、自身の保有するメールアドレスを登録すること。他人のメールアドレスでの登録は認められません。

    （3）(1)の登録事項に変更があった場合、ID保有者は速やかに登録内容の変更を行うこと。

    （4）1人のID保有者による複数のIDの登録および保有は認められません。

    （5）未成年者は、保護者の同意を得てID登録申込みを行うこと。なお、未成年者がID保有者となったときは、保護者の同意があったものとみなします。

  4. ID保有者は対象サービスを利用するにあたり、GakkenIDおよびID登録申込時に登録したパスワードを入力するものとし、学研は、GakkenIDおよびパスワードを利用して行われた行為を、ID保有者による利用とみなすものとします。万一GakkenIDまたはパスワードを盗用されて使用された場合や不正使用された場合であっても、当該行為による責任は学研に故意または重大な過失がない限り、ID保有者が負うものとします。
  5. 学研は、ID保有者が下記のいずれかに該当した場合、当該ID保有者への事前通知を行うことなく当該ID保有者への本サービスおよび対象サービスの全部または一部の利用停止・終了、GakkenIDの抹消、パスワードの削除ならびにID保有者資格の剥奪、損害賠償請求等の必要な対応（法的措置を含みますが、これに限りません）を行うことがあります。なお、当該対応の結果、当該ID保有者に損害や不利益が生じた場合でも、学研は一切の責任を負わないものとします。

    6. （1）ID保有者が本規約または対象サービスの利用時に提示される対象サービスごとの利用規約（以下「対象サービス利用規約」といいます）や諸注意等に違反すると学研が判断する場合またはそのおそれがあると学研が判断する場合

    （2）ID保有者が本サービスまたは対象サービスの運営上不適当と学研が判断する行為を行った場合

  6. ID保有者は、対象サービスを利用する場合、対象サービス利用規約を確認し、同意した上で、対象サービスを利用するものとします。

第4条（ID保有者の禁止事項）

ID保有者は、本サービスおよび対象サービスを利用するにあたり以下の行為または以下に該当するおそれがあると学研が判断する行為を行わないものとします。

    （1）学研、他のID保有者または第三者の著作権、肖像権、その他知的所有権を侵害する行為

    （2）学研、他のID保有者または第三者の財産、プライバシー等を侵害する行為

    （3）学研、他のID保有者または第三者に対する誹謗中傷、迷惑となる行為、不快感を抱かせる行為

    （4）当社の本サービスに係るサーバー等のコンピュータに不正にアクセスし、または有害なコンピュータプログラム、有害なメール等を送信し、もしくは書き込む行為

    （5）本サービスおよび対象サービスを商用目的で利用する行為

    （6）犯罪的行為に結びつく行為

    （7）本サービスおよび対象サービスを通じて入手した情報を、複製、販売その他私的利用の範囲を超えて使用する行為

    （8）本サービスおよび対象サービスの運営を妨げ、あるいは学研または学研の提携先の信頼を毀損するような行為

    （9）営業活動、営利を目的とした情報提供活動を行う行為

    （10）本サービスおよび対象サービスを使用して、学研または学研の提携先を故意に虚偽情報の提供や脅迫によって困惑させる行為

    （11）その他、法律、法令、公序良俗または本規約に違反する行為または違反のおそれのある行為

第5条（不保証）

学研は、本サービスおよび対象サービスにおいて信頼できるサービスや情報をID保有者へ届けるべく努力をしていますが、次の各号について、一切保証しません。本サービスおよび対象サービスの利用に際しては、ID保有者自身が、本サービスおよび対象サービスにおいて提供される情報やサービスの有用性等を判断し、自身の責任でご利用下さい。

    （1）本サービスおよび対象サービス上で提供される全ての情報（対象サービスとして、および対象サービス上に表示される、第三者が管理・運営するリンクサイト内に含まれる一切の情報等を含み、以下同様とします）に関する、有用性、適合性、完全性、正確性、安全性、合法性、最新性等

    （2）本サービスおよび対象サービスを通じて入手できる商品、役務、情報等がID保有者の希望または期待を満たす適切なものであること

    （3）本サービスおよび対象サービスの提供に不具合、エラーまたは障害が生じないこと

    （4）本サービスおよび対象サービスに関連して送信される電子メール、ウェブコンテンツ等に、コンピュータウィルス等の有害なものが含まれていないこと

    （5）本サービスおよび対象サービスの存続または同一性が維持されること

第6条（学研・ID保有者の責任）

  1. 学研は、ID保有者の本サービスおよび対象サービスの利用または利用不能により生じる一切の損害（以下の各号に定める損害、ID保有者もしくは第三者に生じた精神的苦痛、またはその他の金銭的損失を含む不利益を含みますが、これに限りません）に関して学研に故意・重過失がない限り一切の責任を負わないものとします。また、学研が責任を負う場合であっても、学研の責任は直接かつ通常の損害に限られるものとします。

    2. （1）本サービスおよび対象サービスを介して行う、第三者が提供するコンテンツのダウンロード、および、第三者が管理・運営するリンクサイトへのアクセス等の行為により、ID保有者に生じた損害

    （2）学研が相当の安全策を講じたにもかかわらず、本サービスおよび対象サービスの無断改変、本サービスおよび対象サービスに関するデータへの不正アクセス、コンピュータウィルスの混入等の不正行為が行われ、これに起因してID保有者に生じた損害

    （3）通信回線や、コンピュータ等の障害によるシステムまたはサービスの中断、遅滞、中止、データ消失、不正アクセスにより生じた損害

  2. ID保有者の本規約または対象サービス利用規約に違反する行為によって、学研または第三者が損害を被った場合、当該ID保有者は、自己の責任と費用をもって、当該損害の一切を賠償しなければなりません。

第7条（個人情報の取扱）

  1. 当社は、ID保有者のプライバシーを尊重し、ID保有者の個人情報の管理に細心の注意を払います。本サービスの管理・運営に当たって、当社がID保有者から取得した個人情報は、別途定める「個人情報取り扱いについて」（https://gakken-ps.co.jp/privacypolicy/）に従って取り扱うものとします。
  2. 当社は、ID保有者の個人情報を、次の通り共同利用します。

    3. ①共同利用する個人情報の項目

    GakkenIDに登録する一切の個人情報（第8条に定める属性情報を含む）

    ②共同して利用する者の範囲

    当社を含む学研グループを構成する各会社

    ③共同利用における利用目的

    （ⅰ）ECサイト（対象サービスを含む）その他学研グループを構成する各社の事業に関連するサービスの提供、および当該サービス提供に付随する行為（マーケティング、新商品・サービスの企画開発等）

    （ⅱ）学研グループを構成する会社及びその提携先である第三者の商品・サービスの紹介・案内。

    ④個人情報の管理について責任を有する者の名称

    株式会社学研プロダクツサポート

  3. 当社は、ID保有者が対象サービスの利用を停止しまたは対象サービス独自の会員制度から退会等（理由は問いません）したとしても、ID保有者が自ら別途当社が定める手続きに従いGakken IDを抹消しない限り、ID保有者の個人情報を保有し利用し続けることができるものとし、ID保有者は、これを予め同意するものとします。
  4. 当社は、ID保有者の個人情報を共同利用する当社を含む学研グループを構成する各会社の商品発送、商品代金の回収、その他、業務を円滑に進めるために、業務の一部を外部に委託し、業務委託先に対して必要な範囲で個人情報を提供することがあります。この場合、当社はこれらの業務委託先との間で委託された個人情報の取扱いに関する契約の締結をはじめ、当該個人情報の安全管理が図られるよう必要かつ適切な監督を行います。

第8条（属性情報・行動履歴の取得および利用）

  1. 学研は、ID保有者により本サービスおよび対象サービスの利用を通じて提供された年齢や性別、職業、居住地域等の属性情報および対象サービスにおけるID保有者の行動履歴を取得し、対象サービスの提供に必要な範囲で利用すること、および当該属性情報・当該行動履歴に基づき作成された統計データについて、何らの制限なく利用することができることに、ID保有者は予め承諾します。なお、学研は、法律上の根拠に基づき要求された場合およびID保有者の同意を得た場合を除き、ID保有者個人を特定できる情報を第三者に開示しないものとします。
  2. 学研は、ID保有者がGakken IDを用いて対象サービスにログインしこれを利用した場合、個人を特定したうえで、これまでに取得した属性情報および行動履歴を用いて、広告等の配信・表示および対象サービスの提供を行うことができるものとし、ID保有者はこれを予め承諾します。

第9条（対象サービスの変更・中断・終了）

  1. 学研は、ID保有者に事前に通知することなく、本サービスおよび対象サービスの全部または一部を変更、中断、終了することができるものとします。ただし、学研が本サービスおよび対象サービスの全部を終了する場合には、ID保有者に対し、事前に学研の定める方法で通知するものとします。
  2. 本サービスおよび対象サービスの全部または一部の変更・中断・終了に伴い、ID保有者に不利益や損害が発生した場合であっても、学研は一切の責任を負わないものとします。

第10条（反社会的勢力の排除）

ID保有者は、反社会的勢力（暴力団、暴力団員、暴力団員でなくなった時から5年を経過しない者、暴力団準構成員、暴力団関係企業、総会屋等、社会運動等標ぼうゴロまたは特殊知能暴力集団等、その他これらに準ずる者をいいます。）に該当しないこと、また暴力的行為、詐術・脅迫行為、業務妨害行為等違法行為を行わないことを、将来にわたっても表明するものとします。かかる表明に違反した場合には、異議なく本サービスおよび対象サービスの利用停止・終了、GakkenIDの抹消、パスワードの削除ならびにID保有者資格の剥奪、損害賠償請求等の対応（法的措置を含みますが、これに限りません）を受け入れるものとします。

第11条（権利義務の譲渡禁止）

ID保有者は、本サービスおよび対象サービスに関するID保有者としての地位および当該地位に基づく権利義務を、当社が予め承諾した場合を除き、第三者に譲渡しまたは担保に供してはならないものとします。

第12条（準拠法および管轄）

本規約は日本法に基づいて解釈され、本規約に関する紛争の第一審の専属的合意管轄裁判所は、当社本店所在地を管轄する地方裁判所または簡易裁判所を第一審専属管轄裁判所とします。
